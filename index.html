Mengenal HTML, CSS, JavaScript, dan Tailwind CSS

HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat struktur halaman web.

CSS (Cascading Style Sheets) digunakan untuk mengatur tampilan dan layout halaman web agar menarik secara visual.

JavaScript adalah bahasa pemrograman yang membuat halaman web menjadi interaktif dan dinamis.

Tailwind CSS adalah framework CSS utility-first yang memungkinkan Anda membangun desain kustom dengan cepat tanpa menulis CSS dari awal.

Tata Cara Membuat Website Sederhana

  1. Rencanakan struktur website: Tentukan halaman apa saja yang dibutuhkan (home, about, contact).
  2. Buat file HTML dasar: Gunakan tag seperti , , .
  3. Tambahkan konten: Gunakan heading, paragraf, gambar, dan daftar.
  4. Styling dengan CSS: Buat file .css atau gunakan style inline/internal.
  5. Tambahkan interaktivitas dengan JavaScript (opsional).
  6. Uji di browser dan sesuaikan tampilan di berbagai ukuran layar.
